# TalentScout Hiring Assistant Chatbot

## Overview
TalentScout is an AI-powered hiring assistant chatbot designed to automate the initial technical screening process. It engages candidates in a conversational manner, collects essential information, and generates relevant technical questions based on the candidate's tech stack.

## Features
- **Conversational Interface**: Professional and friendly chatbot interaction
- **Information Collection**: Captures 7 key candidate fields including tech stack
- **Smart Question Generation**: Creates 3-5 technical questions per technology with optional coding tasks
- **Context Awareness**: Maintains conversation flow and remembers previous answers
- **Exit Handling**: Gracefully ends conversation with appropriate keywords
- **Data Security**: Masks sensitive information before storage
- **Modular Architecture**: Clean, maintainable code structure
